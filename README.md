# Fa(c_e)Script
![faces_logo](./images/faces_full_logo_background.png "faces_logo")

Fa(c_e)Script（フェイススクリプト）は、顔文字を並べてコードを記述するプログラミング言語です。

## 1. 値
値は、`(o_o)`を1、`(-_-)`を0とする二進数で表されます。
```
(-_-) -> 0
(o_o) -> 1
(o_o)(o_o) -> 3
(o_o)(-_-)(-_-)(o_o) -> 9
```

## 2. 命令と引数
`⅃`を（顔から見て）左手とする顔文字を手続き顔と呼びます。
命令顔と顔文字`L(-.<)`
